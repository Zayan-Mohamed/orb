# GIF Placeholders

This directory will contain screen recording GIFs for the documentation.

## Required GIFs

1. **orb-share-demo.gif**

   - Demonstrates sharing a folder with `orb share`
   - Shows session ID and passcode generation
   - Target size: ~2-3 MB
   - Duration: ~15-20 seconds

2. **orb-connect-demo.gif**

   - Demonstrates connecting with `orb connect`
   - Shows TUI file browser navigation
   - Target size: ~3-4 MB
   - Duration: ~20-25 seconds

3. **orb-complete-demo.gif**
   - Shows complete workflow from share to download
   - Split-screen or sequential view
   - Target size: ~4-5 MB
   - Duration: ~25-30 seconds
